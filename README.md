# FastAPI Multi Product AI API

This example demonstrates how to structure a multi-product AI API using FastAPI. It includes distinct endpoints for different AI functionalities like text summarization, sentiment analysis, and translation. A basic API key verification is implemented as a placeholder for access control, reflecting the concepts of monetization and user management discussed in the article.

## Language

`python`

## How to Run

1. Install dependencies: `pip install fastapi uvicorn`.
2. Save the code as `main.py` and run it: `python main.py`.
3. Access the API at `http://localhost:8000` or the interactive documentation at `http://localhost:8000/docs`.
